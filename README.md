# Raed Hamze — UAE Real Estate Authority

Personal authority site for Raed Hamze, Founder & CEO of Sky Land Middle East (Abu Dhabi) and Luxury Sky Land (Dubai).

Built with React, Vite, TypeScript, Tailwind CSS, and Framer Motion.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Deployed on Vercel. See `vercel.json` for build configuration.
